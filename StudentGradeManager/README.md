# Student Grade Manager

A simple C# console application to manage students and their grades.

## Features

- Add students and their grades
- Calculate each student's average grade
- Display highest and lowest averages
- Remove a student
- Input validation (grades must be between 0 and 100)
- Interactive console menu

## How to Run

1. Make sure [.NET SDK](https://dotnet.microsoft.com/download) is installed
2. Clone or download this project
3. Open terminal in project folder
4. Run the app:

```bash
dotnet run
